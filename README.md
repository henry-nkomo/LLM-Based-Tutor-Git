## File Organisation
Folders: 
Past Papers Chosen - Contains past papers used as tutoring material
Project Files - Contains the soucrce code which are run from the main.py  file

Files:
main.py - The main file for running the program.
requrements.txt - The installation packages required to run the program.
rag_indexing - contains the code used to upload the past papers (teaching material) to the vector database.

## Installation

To setup and run this project do this:
1) Install the packages from the requirements.txt  file.
2) Create an account with Pincone, Claudi and obtain access API keys.
3) Upload the past papers to pinecone, you may follow the approach in the RAG_indexing file 
4) Clone the repo
4) Run the project from the main.py file
